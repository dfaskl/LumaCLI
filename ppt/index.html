Why Java · 为什么是它

用最熟的语言，做最新的事

不用先去换语言，也不用啃框架源码——从课程设计那套技术栈，直接走进 Agent 的世界。

零迁移成本
Java 17 + Maven。写课程设计用的那套东西，刚好就是入场券。
零框架黑盒
不依赖 LangChain。OkHttp + Jackson 手写每一次调用，没有看不懂的魔法。
一份有故事的简历
"我从零写过一个 Agent CLI"——在 27 届的秋招桌上，是稀缺的开场白。
